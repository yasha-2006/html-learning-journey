# HTML Lists

Lists are used to display related items in an organized way.

HTML provides:

1. Unordered Lists (`<ul>`)
2. Ordered Lists (`<ol>`)
3. List Items (`<li>`)

## Unordered List
Displays items with bullets.

## Syntax:

```html
<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>
```

Example:

```html
<ul>
    <li>Python</li>
    <li>Java</li>
    <li>C</li>
</ul>
```

## Ordered List
Displays items with numbers.

## Syntax:

```html
<ol>
    <li>Item 1</li>
    <li>Item 2</li>
</ol>
```

Example:

```html
<ol>
    <li>Wake Up</li>
    <li>Study</li>
    <li>Sleep</li>
</ol>
```

## List Item Tag
The `<li>` tag defines a list item.

Example:
```html
<li>Python</li>
```
## Nested Lists

Lists can contain other lists.

Example:

```html
<ul>
    <li>Programming
        <ul>
            <li>Python</li>
            <li>Java</li>
        </ul>
    </li>
</ul>
```

## Ordered List Types

### Numbers (Default)
```html
<ol type="1">
```
### Uppercase Letters
```html
<ol type="A">
```
### Lowercase Letters
```html
<ol type="a">
```
### Uppercase Roman
```html
<ol type="I">
```
### Lowercase Roman
```html
<ol type="i">
```
## Table

*  `<ul>`-> Unordered List 
*  `<ol>`-> Ordered List 
* `<li>` -> List Item 
## Revision

- `<ul>` → Bulleted list
- `<ol>` → Numbered list
- `<li>` → List item
- Nested lists are allowed
- Ordered lists support numbering styles
