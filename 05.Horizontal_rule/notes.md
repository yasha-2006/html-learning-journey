# HTML Horizontal Rule Tag

The `<hr>` tag is used to create a horizontal line on a webpage.
It is commonly used to separate sections of content.

## Syntax

```html
<hr>
```
Example:

```html
<h1>About Me</h1>

<p>I am learning HTML.</p>

<hr>

<p>I am also learning CSS.</p>
```

## Important Points

- `<hr>` stands for Horizontal Rule.
- It creates a horizontal line.
- It is an empty tag.
- No closing tag is required.
- Used to separate content sections.

## Real-Life uses
 1. Blog Posts
 2. Resume
 3. Portfolio Website


## Difference Between `<br>` and `<hr>`

 `<br>`                        | `<hr>` 
 * Moves content to next line  | Creates horizontal line 
 * No visible line             | Visible separator 
 * Used inside text            | Used between sections
