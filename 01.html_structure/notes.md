HTML?
* It stands for HyperYext Markup Language.
* It is the standard language used to creata web pages.

Explanation of Tags:

1. <!DOCTYPE html>
  * Declares that the documents is an HTML5 document.
  * Must be the first line of every HTML page.

2. <html>
  * Root element of an HTML document.
  * All HTML content is placed inside this tag.

3. <head>
  * Contains information about the webpage.
  * Not visible on the webpage itself.  

 Examples:
  * Title
  * CSS links
  * Meta tags

4. <title>
  * Sets the title shown on the browser tab.

 Example:
   <title>My Website</title>

5. <body>
  * Contains all visible content of the webpage.

 Examples:
 * Headings
 * Paragraphs
 * Images
 * Links
 * Tables
 * Forms
