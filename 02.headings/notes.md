What are Headings?

Headings are used to define titles and subtitles on a webpage.

HTML provides six heading levels:

<h1> - Largest and most important heading
<h2> - Second-level heading
<h3> - Third-level heading
<h4> - Fourth-level heading
<h5> - Fifth-level heading
<h6> - Smallest heading

Syntax: 

<h1>Main Heading</h1>
<h2>Sub Heading</h2>
<h3>Section Heading</h3>
<h4>Topic Heading</h4>
<h5>Small Heading</h5>
<h6>Smallest Heading</h6>

Advantages of Headings:

1. Organize webpage content.
2. Improve readability.
3. Help search engines understand page structure.
4. Improve accessibility for screen readers.
5. Make websites easier to navigate.
