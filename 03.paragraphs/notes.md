 What is a Paragraph?
A paragraph in HTML is defined using the `<p>` tag.

Syntax:
<p>This is a paragraph.</p>


Example:

```html
<p>HTML is used to create web pages.</p>
<p>CSS is used to style web pages.</p>
```

## Features of Paragraphs

1. Automatic Spacing
   Browsers automatically add space before and after paragraphs.

 2. Extra Spaces are Ignored
    HTML ignores multiple spaces.

 3. Multiple Lines are Ignored
    HTML displays text as a single line unless special tags are used.
