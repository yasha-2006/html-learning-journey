# HTML Images

Images are displayed in HTML using the `<img>` tag.
Images make webpages attractive and informative.
## The `<img>` Tag

The `<img>` tag is used to display images.

## Syntax

```html
<img src="image.jpg" alt="Description">
```
Example:

```html
<img src="profile.jpg" alt="Profile Picture">
```
## The src Attribute

`src` stands for Source.
It specifies the path of the image file.

## The alt Attribute

`alt` stands for Alternative Text.
It is displayed when the image cannot be loaded.

## Image Path type

### Relative Path

```html
<img src="images/profile.jpg">
```
Uses images from your project folder.

### Absolute Path

```html
<img src="https://example.com/image.jpg">
```
Uses images from the internet.

## Table

* img -> Display image
*  src -> Image location 
*  alt -> Alternative text
* width -> Image width 
* height -> Image height
