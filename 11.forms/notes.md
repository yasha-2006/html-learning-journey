# HTML Forms
Forms are used to collect user input.

Examples:

- Login Form
- Registration Form
- Feedback Form
- Contact Form

Forms are created using the `<form>` tag.

## Basic Syntax
```html
<form>

    <input type="text">

</form>
```
## Form Tag
The `<form>` tag creates a form.
```html
<form>
</form>
```
## Text Input
Used to enter text.
```html
<input type="text">
```
Example:
```html
Name:
<input type="text">
```
## Email Input
Used for email addresses.
```html
<input type="email">
```
## Password Input
Used for passwords.
```html
<input type="password">
```
Characters appear hidden.
## Number Input
Used for numbers.
```html
<input type="number">
```
## Radio Buttons
Allows selecting one option.
```html
<input type="radio" name="gender"> Male

<input type="radio" name="gender"> Female
```
## Checkboxes
Allows selecting multiple options.
```html
<input type="checkbox"> HTML

<input type="checkbox"> CSS
```
## Dropdown List
Created using `<select>`.
```html
<select>

    <option>AIML</option>
    <option>CSE</option>

</select>
```
## Text Area
Used for multi-line text.
```html
<textarea rows="4" cols="30">
</textarea>
```
## Submit Button
Submits the form.
```html
<input type="submit">
```

## Common Input Types

*  text -> Text input 
*  email -> Email input 
*  password -> Password input 
*  number -> Numeric input 
*  radio -> Single choice 
*  checkbox -> Multiple choices 
*  submit -> Submit form 

## Revision

- `<form>` → Form
- `<input>` → Input field
- `type="text"` → Text
- `type="email"` → Email
- `type="password"` → Password
- `type="radio"` → Single choice
- `type="checkbox"` → Multiple choice
- `<textarea>` → Multi-line text
- `<select>` → Dropdown
- `type="submit"` → Submit button
