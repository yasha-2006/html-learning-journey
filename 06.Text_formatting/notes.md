# HTML Text Formatting

HTML provides formatting tags to change the appearance and meaning of text.
These tags help make content more readable and attractive.

## Bold Text
`<b>` Tag -> Displays text in bold.

```html
<b>Bold Text</b>
```


## Important Text
### `<strong>` Tag
Indicates important text.

```html
<strong>Important Text</strong>
```

## Italic Text
### `<i>` Tag

Displays text in italic style.

```html
<i>Italic Text</i>
```

## Emphasized Text
### `<em>` Tag

Displays emphasized text.

```html
<em>Emphasized Text</em>
```
## Underlined Text
### `<u>` Tag

Underlines text.

```html
<u>Underlined Text</u>
```

## Highlighted Text
### `<mark>` Tag

Highlights text.

```html
<mark>Highlighted Text</mark>
```

## Small Text
### `<small>` Tag

Displays smaller text.

```html
<small>Small Text</small>
```

## Subscript Text
### `<sub>` Tag

Displays text below the normal line.

```html
H<sub>2</sub>O
```

## Superscript Text
### `<sup>` Tag

Displays text above the normal line.

```html
x<sup>2</sup>
```

## Deleted Text
### `<del>` Tag

Shows deleted text with a strike-through line.

```html
<del>Old Price</del>
```

## Summary Table

- `<b>` → Bold
- `<strong>` → Important
- `<i>` → Italic
- `<em>` → Emphasized
- `<u>` → Underline
- `<mark>` → Highlight
- `<small>` → Small text
- `<sub>` → Subscript
- `<sup>` → Superscript
- `<del>` → Deleted text

## Difference Between Similar Tags

### `<b>` vs `<strong>`

   `<b>`                       | `<strong>` 
  * Bold appearance            | * Important meaning 
  * Visual formatting          | * Semantic importance 

### `<i>` vs `<em>`

 `<i>`                     | `<em>` 
 *  Italic appearance      | * Emphasized meaning
